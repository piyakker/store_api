# Store API

be able to get all products' data

## feature

+ get all products' data
+ search for specific product with different filters
+ sort products in different kind of orders
+ choose which property you want to present
+ be able to choose different pages

## Usage

1. Clone this project from Github via code below in the Terminal.

```
git clone https://github.com/piyakker/store_api.git
```

2. Go into the directory of the cloned project

```
cd store_api
```

3. Install dependencies via npm
  
```
npm install
```

4. run the script and once the URL is shown, the server will be able to use.

```
npm run dev 
```

```
listening on http://localhost:3000
```
